# Code Names

Play it here: https://ghelouis.github.io/codenames/

Forked from https://github.com/jhil/kodenames

[wiktionary](https://fr.wiktionary.org/wiki/Wiktionnaire:Liste_de_1750_mots_fran%C3%A7ais_les_plus_courants)
